# Galaga game

조작방법 : 모바일(디바이스 기울기)  PC(방향키)
<br>play : http://yechanball.github.io/game

## 게임 설정 기능

- `설정` 버튼으로 설정 패널 열기/닫기
- 조절 항목
  - 플레이어 이동 속도
  - 적 이동 속도
  - 총알 속도
  - 적 등장 간격
  - 자동 발사 간격
  - 기울기 민감도(Deadzone, 낮을수록 민감)
- `기본값 복원` 버튼 지원
- 설정 값은 브라우저 `localStorage`에 저장되어 다음 실행에도 유지됨

## 플레이 테스트 체크리스트

- [ ] 화살표 키(좌/우/상/하) 입력으로 비행선이 정상 이동한다.
- [ ] 총알이 일정 주기로 자동 발사되고, 적과 충돌 시 점수가 증가한다.
- [ ] 적과 비행선이 충돌하면 배경 dim + 충돌 이펙트 후 게임 오버 오버레이가 표시된다.
- [ ] 게임 오버 화면의 `다시 시작` 버튼으로 즉시 새 게임이 시작된다.
- [ ] GravitySensor 미지원 환경(일반 데스크톱 브라우저)에서도 오류 없이 실행된다.
- [ ] 설정 패널에서 속도/민감도 슬라이더를 조절하면 즉시 반영된다.
- [ ] 페이지를 새로고침해도 마지막 설정 값이 유지된다.
